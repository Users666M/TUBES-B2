# 🎵 Spotify Clone - Project Tubes B2

Aplikasi pemutar musik berbasis web sederhana yang terinspirasi dari Spotify. Project ini dibuat sebagai tugas besar (Tubes) untuk mata kuliah Proyek Pemrograman Web.

## 🌟 Fitur Utama
* **Autentikasi User**: Fitur Registrasi dan Login untuk keamanan akun.
* **Manajemen Musik**: Menampilkan daftar lagu pilihan dengan tampilan kartu.
* **Dashboard Admin**: Halaman khusus untuk mengelola data lagu dan user.
* **Responsive Design**: Tampilan yang menyesuaikan perangkat (Mobile & Desktop) menggunakan CSS Modern/Bootstrap.

## 🚀 Teknologi yang Digunakan
* **Backend**: Node.js & Express.js
* **Database**: MySQL (Hosted on Railway)
* **Frontend Engine**: EJS (Embedded JavaScript)
* **Styling**: CSS / Bootstrap
* **Deployment**: Railway App & GitHub
